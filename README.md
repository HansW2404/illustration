# Hans Wang - Portfolio Website

A creative portfolio website showcasing UI/UX development and illustration work. Features smooth animations, custom cursor effects, interactive background canvas, and scroll-triggered animations for an engaging user experience.

## Author

Hans Wang - UI/UX Developer & Illustrator

## Folder Structure

```
portfolio/
├── index.html                 # Main HTML structure
├── style.css                  # Custom styles and animations
├── script.js                  # Interactive features and animations
└── assets/
    └── Hans Wang - resume.pdf # Resume download
```

## Features

- Canvas background with floating shapes
- Smooth scroll navigation
- Scroll-triggered section animations
- Parallax effects on hero section
- Responsive design

Built with vanilla HTML, CSS, and JavaScript.
